# 中秋国庆贺卡 🌕🇨🇳

这是一个简单的网页项目，允许您在中秋和国庆主题之间切换贺卡 🎉。通过点击头像 🖼️，切换主题并显示不同的祝福语 🎆。

## 特点 ✨

- 单击头像即可在中秋和国庆主题之间切换。
- 欣赏设计精美的贺卡，带有动画效果 💌。
- 与朋友和家人分享温馨的祝福 🌟。
